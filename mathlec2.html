I. Function

DEFINITION Function:

A function from a set D to a set Y is a rule that assigns a unique (single) element $f(x)\in Y$ to each element $x\in D$

Definition: Let X, Y be non-empty sets, and let $f: X \rightarrow Y$ be a relation from X into Y

How to show that $f(x)$ is a function?

  • 1- Algebraically: By showing that: for all $x\in X$ there exists a unique $y\in Y$ : $y=f(x)$
  • 2- Graphically: "V.L.T.” (Vertical Line Test): "A relation f is a function if and only if its graph intersects each vertical line at most once"

Reminder:

A function $f$ is a rule that assigns exactly one element $y$ in a set $B$ to each element $x$ in a set $A$. In this case, we write $y=f(x)$.

We call the set $A$ the domain of $f$. The set of all values $f(x)$ in $B$ is called the range of $f$. That is, the range of $f$ is $\{f(x)|x\in A\}$. Unless explicitly stated otherwise, the domain of a function $f$ is the largest set of real numbers for which the function is defined. We refer to $x$ as the independent variable and to $y$ as the dependent variable.

Once $f$ defines a function, $X=D_{f}=$ Domain & $Y=R_{f}=$ Range

  • $D_{f}=$ the set of all possible input values "$x$" that we can substitute in the given function $f(x)$.
  • $D_{f}=$ the projection of the curve of $f$ on the x-axis.
  • $R_{f}=$ the set of all output values "$y$" "images" "outcomes" we obtained after substituting with the inputs "$x$" in $f(x)$.
  • $R_{f}=$ the projection of the curve of $f$ on the y-axis.

Examples:

  • Example 1: The relation $x^{2}+y^{2}=9$ is NOT a function using any of the two methods. Because: @ $x=0$ y does not have a unique value, but it has two values $\pm3$.
  • Example 2: The relation $f(x)=2x+1$ is a function using the previous two methods.

II. One-to-one functions & Inverse of a Function

DEFINITION One-to-One Function:

A function $f(x)$ is one-to-one on a domain D if $f(x_{1})\ne f(x_{2})$ whenever $x_{1}\ne x_{2}$ in D.

Equivalently, $f(x_{1})=f(x_{2})$ implies $x_{1}=x_{2}$

How to show that $f$ is a 1-1 function?

  • 1- Algebraically: We assume $f(x_{1})=f(x_{2})$, then we check if $x_{1}=x_{2}$?
  • 2- Graphically: “H.L.T.” (Horizontal Line Test): "A function f is a 1-1 if and only if its graph intersects each horizontal line at most once"

Exercises (One-to-one):

Check algebraically and graphically if the following functions are one-to-one or not.

  1. $f(x)=2x+3$
  2. $f(x)=x^{2}$
  3. $f(x)=(3x+5)^{1/3}+3$

Solution (algebraically) for 1:

Assume: $f(x_{1})=f(x_{2})$,
Then, $2x_{1}+3=2x_{2}+3$,
Therefore, $x_{1}=x_{2}$,
Hence $f(x)$ is one - to - one.

Using the horizontal line test, we see that $y=x^{3}$ and $y=\sqrt{x}$ are one-to-one on their domains $(-\infty,\infty)$ and $[0,\infty)$, but $y=x^{2}$ and $y=\sin x$ are not one-to-one on their domains $(-\infty,\infty)$.

Inverse Function Definition:

If $f$ is a function “with domain $D_f$ and range $R_f$” , we say that a function $g$ “with domain $R_f$ and range $D_f$” is the valid inverse of the function $f$ if and only if:

  1. The function $f$ is a 1-1.
  2. $g(f(x)) = x$, for all $x$ in $D_f$ and $f(g(y)) = y$, for all $y$ in $R_f$.

Notation: Usually $g$ is denoted by $f^{-1}$. So we write $f^{-1}(f(x)) = x$ and $f(f^{-1}(y)) = y$ (cancellation rule).

It is clear that: $x = f^{-1}(y)$, $y = f(x)$, $D_{f^{-1}} = R_f$, $R_{f^{-1}} = D_f$

Definition: If $f$ is a 1-1 function, then $f$ is said to have an inverse function denoted by $f^{-1}$.

How to make sure that the obtained inverse is correct?

We have to prove that:

  1. The function $f$ is 1-1, then we find its inverse.
  2. Checkup: $f^{-1}(f(x)) = x$, for all $x$ in Domain of $f$ and $f(f^{-1}(x)) = x$, for all $x$ in $D_{f^{-1}} = R_f$.

How to sketch the graph of $f^{-1}$?

“If you know the graph of $f$”

  1. Draw the dashed line $y=x$
  2. Reflect the graph of $f$ on this line $y=x$ to get the graph of $f^{-1}$.

How to find $f^{-1}(x)$ if you know $f(x)$?

Find and graph the inverse function of each of the following functions:

  1. $f(x) = 0.5x + 1$
  2. $f(x) = x^2$ on $D_f = [0, \infty[$
  3. $f(x) = (x+5)^{1/2} + 3$ (Homework)
  4. $f(x) = 2 + (2x-4)^{-1}$ (What do you notice?)

First you must show (from the graph) that the given functions are One-to-One

Another Solution of #1:

(i) Since $f(x) = 0.5x + 1$ for all $x$
(ii) Therefore $f(\_)= (1/2) \_ + 1$
(iii) i.e., $f(f^{-1}(x)) = (1/2)f^{-1}(x) +1$
(iv) We know that: $f(f^{-1}(x)) = x$
(v) We get, $x = (1/2)f^{-1}(x) +1$
(vi) Solving for $f^{-1}(x)$ we get $2x = f^{-1}(x) + 2$
Hence, $f^{-1}(x) = 2x-2$

Points to remember for a function:

  1. How to check if a function is 1-1 or not (by using the algebraic test or the H.L.T.)
  2. If the function is 1-1, then:
    • Find its inverse function.
    • Make sure that it is a correct inverse.
    • Sketch the inverse.
    • Identify the domain and range of the inverse.
    • Put a name for the existed inverse function.
  3. If the function is Not 1-1, then what should we do so that it becomes 1-1 and then to proceed with its inverse and so on.

Summary of Domains

Understanding the domain of a function is crucial for defining where the function is valid and behaves as expected. Here's a quick reference for common function types:

Function Type Typical Domain Notes
Polynomial Functions (e.g., $y=x^2+3x-1$) $\mathbb{R}$ or $(-\infty, \infty)$ All real numbers. No restrictions.
Rational Functions (e.g., $y=1/x$, $y=(x+2)/(x-3)$) All real numbers except where the denominator is zero. Set denominator $\ne 0$ and solve for $x$.
Square Root (or even root) Functions (e.g., $y=\sqrt{x}$) All real numbers where the expression under the root is non-negative ($\ge 0$). Set expression under root $\ge 0$ and solve for $x$.
Cube Root (or odd root) Functions (e.g., $y=\sqrt[3]{x}$) $\mathbb{R}$ or $(-\infty, \infty)$ All real numbers. No restrictions.
Logarithmic Functions (e.g., $y=\ln(x)$, $y=\log_{10}(x)$) All real numbers where the argument of the logarithm is strictly positive ($> 0$). Set argument of log $> 0$ and solve for $x$.
Exponential Functions (e.g., $y=e^x$, $y=2^x$) $\mathbb{R}$ or $(-\infty, \infty)$ All real numbers. No restrictions.
Trigonometric Functions (e.g., $y=\sin(x)$, $y=\cos(x)$) $\mathbb{R}$ or $(-\infty, \infty)$ For $\tan(x)$, $\sec(x)$, etc., exclude values where denominator (cosine) is zero.
Inverse Trigonometric Functions (e.g., $y=\arcsin(x)$) Specific intervals, depends on the function. For $\arcsin(x)$ and $\arccos(x)$, the domain is $[-1, 1]$.
