a. Logarithmic Differentiation

The derivatives of functions of the form $y = f(x)^{g(x)}$ where both $f(x)$ and $g(x)$ are non-constant functions cannot be found using the simple power rule or exponential rule. Logarithmic differentiation is used for these types of functions, as well as products or quotients involving many terms.

Steps:

  1. Take the natural logarithm on both sides of the equation.
  2. Use logarithm properties to simplify the right side of the equation.
  3. Differentiate both sides with respect to $x$.
  4. Solve for $\frac{dy}{dx}$.

Example 1: Find $y^{\prime}$ if $y=x^{\sin x}$.

Solution: Given $y=x^{\sin x}$. Take $\ln$ on both sides: $\ln y = \ln(x^{\sin x}) \Rightarrow \ln y = (\sin x)(\ln x)$ Differentiate implicitly with respect to $x$: $\frac{1}{y}y^{\prime} = (\sin x)\left(\frac{1}{x}\right) + (\cos x)(\ln x)$ $y^{\prime} = y\left[\frac{\sin x}{x} + (\cos x)(\ln x)\right]$ Substitute $y=x^{\sin x}$: $y^{\prime} = x^{\sin x}\left[\frac{\sin x}{x} + (\cos x)(\ln x)\right]$

Example 2: Find $y^{\prime}$ if $y=(\ln x)^{\tan x}$.

Solution: Given $y=(\ln x)^{\tan x}$. Take $\ln$ on both sides: $\ln y = \ln((\ln x)^{\tan x}) \Rightarrow \ln y = (\tan x)(\ln(\ln x))$ Differentiate implicitly with respect to $x$: $\frac{1}{y}y^{\prime} = (\tan x)\left(\frac{1}{\ln x}\cdot\frac{1}{x}\right) + (\sec^{2}x)(\ln(\ln x))$ $y^{\prime} = y\left[\frac{\tan x}{x\ln x} + (\sec^{2}x)(\ln(\ln x))\right]$ Substitute $y=(\ln x)^{\tan x}$: $y^{\prime} = (\ln x)^{\tan x}\left[\frac{\tan x}{x\ln x} + (\sec^{2}x)(\ln(\ln x))\right]$

c. Implicit Differentiation

An equation in $x$ and $y$ is said to define $y$ implicitly as a function of $x$ if the graph of the equation is also the graph of the function. To find $\frac{dy}{dx}$ for such relations, we use implicit differentiation.

Steps:

  1. Differentiate both sides of the equation with respect to $x$. Remember that $y$ is a function of $x$, so apply the Chain Rule when differentiating terms involving $y$.
  2. Solve the resulting equation for $\frac{dy}{dx}$.

Example 1: Find $y^{\prime}$ for $x^{2}+y^{2}=16$.

Solution: Given $x^{2}+y^{2}=16$. Differentiate with respect to $x$: $2x+2y\frac{dy}{dx}=0$ $2y\frac{dy}{dx}=-2x$ $\frac{dy}{dx}=-\frac{x}{y}$

Example 2: Find the equation of the tangent line to the curve $x^{2}+xy+y^{2}=7$ at the point $(1,2)$.

Solution: First, find the derivative $\frac{dy}{dx}$ using implicit differentiation: $2x + (1\cdot y + x\cdot\frac{dy}{dx}) + 2y\frac{dy}{dx} = 0$ $2x+y+x\frac{dy}{dx}+2y\frac{dy}{dx}=0$ $(x+2y)\frac{dy}{dx}=-2x-y$ $\frac{dy}{dx}=\frac{-2x-y}{x+2y}$ Now, evaluate the slope at $(1,2)$: $m = \left.\frac{dy}{dx}\right|_{(1,2)} = \frac{-2(1)-2}{1+2(2)}=\frac{-4}{5}$ The equation of the tangent line (point-slope form $y-y_1=m(x-x_1)$): $y-2=-\frac{4}{5}(x-1)$ $5(y-2)=-4(x-1)$ $5y-10=-4x+4$ $4x+5y=14$

d. Linearization of a Function

The linearization of $f(x)$ at $x=a$ is the equation of the tangent line to the curve $y=f(x)$ at the point $(a, f(a))$. The linearization is given by the formula: $$L(x)=f(a)+f^{\prime}(a)(x-a)$$

Example 1: Find the linearization of $f(x)=\sqrt{x}$ at $x=4$.

Solution: Given $f(x)=\sqrt{x}$ and $a=4$. First, find $f(a)$ and $f^{\prime}(a)$: $f(4)=\sqrt{4}=2$ $f^{\prime}(x)=\frac{1}{2\sqrt{x}} \Rightarrow f^{\prime}(4)=\frac{1}{2\sqrt{4}}=\frac{1}{4}$ Now use the linearization formula: $L(x)=f(a)+f^{\prime}(a)(x-a)$ $L(x)=2+\frac{1}{4}(x-4)$ $L(x)=2+\frac{1}{4}x-1$ $L(x)=\frac{1}{4}x+1$

Example 2: Use linearization to approximate $\sqrt{4.1}$.

Solution: Using the linearization from Example 1, $L(x)=\frac{1}{4}x+1$. We want to approximate $\sqrt{4.1}$, so $x=4.1$. $\sqrt{4.1} \approx L(4.1) = \frac{1}{4}(4.1)+1 = 1.025+1 = 2.025$

The actual value of $\sqrt{4.1}$ is approximately $2.02484567...$, so the approximation is very close.
