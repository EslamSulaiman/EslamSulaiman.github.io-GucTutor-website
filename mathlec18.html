Integration by Substitution (The Reverse Chain Rule)

This is a powerful technique for integrating functions that are products of functions where one function is the derivative of another (or a multiple of it), or for integrals of composite functions.

The method works when the integrand can be written in the form $f(g(x))g^{\prime}(x)$ or $f(g(x)) \cdot k \cdot g'(x)$ for some constant $k$.

Theorem (The Substitution Rule):

If $u=g(x)$ is a differentiable function whose range is an interval $I$, and $f$ is continuous on $I$, then $$\int f(g(x))g^{\prime}(x)dx=\int f(u)du$$ This is also called the "Change of Variables" method.

How to use the substitution method:

  1. Choose a suitable substitution $u=g(x)$. Usually, $u$ is chosen to be the "inner" function of a composite function or a part of the integrand whose derivative is also present (or a constant multiple of it).
  2. Calculate $du = g^{\prime}(x)dx$.
  3. Substitute $u$ and $du$ into the integral to express it entirely in terms of $u$.
  4. Evaluate the integral with respect to $u$.
  5. Replace $u$ with $g(x)$ to express the answer in terms of $x$.

Integrals of Basic Trigonometric Functions by Substitution

1. $\int \tan x dx = \int \frac{\sin x}{\cos x} dx$
Let $u=\cos x \Rightarrow du=-\sin x dx \Rightarrow -du=\sin x dx$.
$\int \frac{-du}{u}=-\int \frac{1}{u}du=-\ln|u|+C=-\ln|\cos x|+C=\ln|\sec x|+C$.

2. $\int \cot x dx = \int \frac{\cos x}{\sin x} dx$
Let $u=\sin x \Rightarrow du=\cos x dx$.
$\int \frac{du}{u}=\ln|u|+C=\ln|\sin x|+C$.

3. $\int \sec x dx = \int \sec x \frac{\sec x+\tan x}{\sec x+\tan x} dx = \int \frac{\sec^{2}x+\sec x \tan x}{\sec x+\tan x} dx$
Let $u=\sec x+\tan x \Rightarrow du=(\sec x \tan x+\sec^{2}x)dx$.
$\int \frac{du}{u}=\ln|u|+C=\ln|\sec x+\tan x|+C$.

4. $\int \csc x dx = \int \csc x \frac{\csc x+\cot x}{\csc x+\cot x} dx = \int \frac{\csc^{2}x+\csc x \cot x}{\csc x+\cot x} dx$
Let $u=\csc x+\cot x \Rightarrow du=(-\csc x \cot x-\csc^{2}x)dx \Rightarrow -du=(\csc x \cot x+\csc^{2}x)dx$.
$\int \frac{-du}{u}=-\ln|u|+C=-\ln|\csc x+\cot x|+C=\ln|\csc x-\cot x|+C$.

Formulas (Integrals by Substitution)

  • $\int \tan u du=\ln|\sec u|+C$
  • $\int \cot u du=\ln|\sin u|+C$
  • $\int \sec u du=\ln|\sec u+\tan u|+C$
  • $\int \csc u du=\ln|\csc u-\cot u|+C$

Examples on Integration by Substitution

Evaluate the following integrals:

1. $\int (x^{2}+1)^{5}(2x) dx$
Let $u=x^{2}+1 \Rightarrow du=2x dx$.
$\int u^{5}du=\frac{u^{6}}{6}+C=\frac{(x^{2}+1)^{6}}{6}+C$.

2. $\int \sqrt{1+x^{2}}(2x) dx$
Let $u=1+x^{2} \Rightarrow du=2x dx$.
$\int \sqrt{u}du=\int u^{1/2}du=\frac{u^{3/2}}{3/2}+C=\frac{2}{3}(1+x^{2})^{3/2}+C$.

3. $\int \sqrt{4x-1} dx$
Let $u=4x-1 \Rightarrow du=4 dx \Rightarrow dx=\frac{1}{4}du$.
$\int \sqrt{u}\frac{1}{4}du=\frac{1}{4}\int u^{1/2}du=\frac{1}{4}\frac{u^{3/2}}{3/2}+C=\frac{1}{6}(4x-1)^{3/2}+C$.

4. $\int \sec(2x+1)\tan(2x+1) dx$
Let $u=2x+1 \Rightarrow du=2 dx \Rightarrow dx=\frac{1}{2}du$.
$\int \sec u \tan u \frac{1}{2}du=\frac{1}{2}\int \sec u \tan u du=\frac{1}{2}\sec u+C=\frac{1}{2}\sec(2x+1)+C$.

5. $\int (3\sin^{2}x \cos x) dx$
Let $u=\sin x \Rightarrow du=\cos x dx$.
$\int 3u^{2}du=u^{3}+C=\sin^{3}x+C$.

6. $\int \frac{(\ln x)^{3}}{x} dx$
Let $u=\ln x \Rightarrow du=\frac{1}{x} dx$.
$\int u^{3}du=\frac{u^{4}}{4}+C=\frac{(\ln x)^{4}}{4}+C$.

7. $\int x e^{-x^{2}} dx$
Let $u=-x^{2} \Rightarrow du=-2x dx \Rightarrow x dx=-\frac{1}{2}du$.
$\int e^{u}(-\frac{1}{2})du=-\frac{1}{2}\int e^{u}du=-\frac{1}{2}e^{u}+C=-\frac{1}{2}e^{-x^{2}}+C$.

8. $\int \frac{e^{\sqrt{x}}}{\sqrt{x}} dx$
Let $u=\sqrt{x} \Rightarrow du=\frac{1}{2\sqrt{x}} dx \Rightarrow 2du=\frac{1}{\sqrt{x}} dx$.
$\int e^{u}(2)du=2\int e^{u}du=2e^{u}+C=2e^{\sqrt{x}}+C$.

9. $\int \frac{x}{x^{2}+1} dx$
Let $u=x^{2}+1 \Rightarrow du=2x dx \Rightarrow x dx=\frac{1}{2}du$.
$\int \frac{1}{u}\frac{1}{2}du=\frac{1}{2}\int \frac{1}{u}du=\frac{1}{2}\ln|u|+C=\frac{1}{2}\ln(x^{2}+1)+C$.

10. $\int \frac{x}{\sqrt{x^{2}+1}} dx$
Let $u=x^{2}+1 \Rightarrow du=2x dx \Rightarrow x dx=\frac{1}{2}du$.
$\int \frac{1}{\sqrt{u}}\frac{1}{2}du=\frac{1}{2}\int u^{-1/2}du=\frac{1}{2}\frac{u^{1/2}}{1/2}+C=\sqrt{u}+C=\sqrt{x^{2}+1}+C$.

11. $\int \frac{e^{\tan^{-1}x}}{1+x^{2}} dx$
Let $u=\tan^{-1}x \Rightarrow du=\frac{1}{1+x^{2}} dx$.
$\int e^{u}du=e^{u}+C=e^{\tan^{-1}x}+C$.

12. $\int \frac{1}{1+\sqrt{x}} dx$
Let $u=1+\sqrt{x} \Rightarrow du=\frac{1}{2\sqrt{x}} dx \Rightarrow 2\sqrt{x} du=dx$.
Since $\sqrt{x}=u-1$, we have $dx=2(u-1)du$.
$\int \frac{1}{u}2(u-1)du=2\int \frac{u-1}{u}du=2\int (1-\frac{1}{u})du=2(u-\ln|u|)+C=2(1+\sqrt{x}-\ln|1+\sqrt{x}|)+C$.

13. $\int \frac{dx}{x\ln x}$
Let $u=\ln x \Rightarrow du=\frac{1}{x} dx$.
$\int \frac{1}{u}du=\ln|u|+C=\ln|\ln x|+C$.
