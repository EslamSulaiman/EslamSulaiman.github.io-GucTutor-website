1. The Concept "Definition" of the Derivative

For a function $f(x)$, we define the derivative of this function at any $x$ in $D_f$ by: $$f^{\prime}(x)=\lim_{h\rightarrow0}\frac{f(x+h)-f(x)}{h}$$ if the limit exists.

Note: If the limit Does Not Exist or equals to $\pm\infty$, then we say that the function "is not differentiable" or "has no derivative" at $x$.

The derivative is sometimes denoted by: $y^{\prime}=\frac{dy}{dx}$ or $D_x f(x)=f^{\prime}(x)=\frac{df}{dx}$.

Notice that: Since $f$ is a function of $x$, then its derivative $f'$ is also a function of $x$.

Example 1: Use the definition of derivative to find $f^{\prime}(x)$, if $f(x)=x^{2}$.

Solution: Given $f(x)=x^{2}$, then $f(x+h)=(x+h)^{2}$. $$f^{\prime}(x)=\lim_{h\rightarrow0}\frac{f(x+h)-f(x)}{h}$$ $$=\lim_{h\rightarrow0}\frac{(x+h)^{2}-x^{2}}{h}$$ $$=\lim_{h\rightarrow0}\frac{x^{2}+2xh+h^{2}-x^{2}}{h}$$ $$=\lim_{h\rightarrow0}\frac{2xh+h^{2}}{h}$$ $$=\lim_{h\rightarrow0}\frac{h(2x+h)}{h}$$ $$=\lim_{h\rightarrow0}(2x+h)=2x$$ So, $f^{\prime}(x)=2x$.

3. Derivative of Square Root Function

Example 2: Use the definition of derivative to find $f^{\prime}(x)$, if $f(x)=\sqrt{x}$.

Solution: Given $f(x)=\sqrt{x}$, then $f(x+h)=\sqrt{x+h}$. $$f^{\prime}(x)=\lim_{h\rightarrow0}\frac{f(x+h)-f(x)}{h}$$ $$=\lim_{h\rightarrow0}\frac{\sqrt{x+h}-\sqrt{x}}{h}$$ Multiply by the conjugate: $$=\lim_{h\rightarrow0}\frac{\sqrt{x+h}-\sqrt{x}}{h}\cdot\frac{\sqrt{x+h}+\sqrt{x}}{\sqrt{x+h}+\sqrt{x}}$$ $$=\lim_{h\rightarrow0}\frac{(x+h)-(x)}{h(\sqrt{x+h}+\sqrt{x})}$$ $$=\lim_{h\rightarrow0}\frac{h}{h(\sqrt{x+h}+\sqrt{x})}$$ $$=\lim_{h\rightarrow0}\frac{1}{\sqrt{x+h}+\sqrt{x}}$$ $$=\frac{1}{\sqrt{x+0}+\sqrt{x}}=\frac{1}{2\sqrt{x}}$$ So, $f^{\prime}(x)=\frac{1}{2\sqrt{x}}$.

4. Derivative of Absolute Value Function

Example 3: Use the definition of derivative to find $f^{\prime}(x)$ for $f(x)=|x|$ for any $x\ne0$.

Solution: We now find the derivative of $|x|$ at any $x\ne0$. $$f^{\prime}(x)=\lim_{h\rightarrow0}\frac{f(x+h)-f(x)}{h}$$ Since $x \ne 0$:

  • If $x>0$, then $|x|=x$ and for small $h$, $|x+h|=x+h$. $f^{\prime}(x)=\lim_{h\rightarrow0}\frac{(x+h)-x}{h}=\lim_{h\rightarrow0}\frac{h}{h}=1$.
  • If $x<0$, then $|x|=-x$ and for small $h$, $|x+h|=-(x+h)$. $f^{\prime}(x)=\lim_{h\rightarrow0}\frac{-(x+h)-(-x)}{h}=\lim_{h\rightarrow0}\frac{-x-h+x}{h}=\lim_{h\rightarrow0}\frac{-h}{h}=-1$.
Therefore, $f^{\prime}(x)=\frac{x}{|x|}$. Note that at $x=0$, the derivative does not exist as the left and right limits of the difference quotient are $-1$ and $1$ respectively.
