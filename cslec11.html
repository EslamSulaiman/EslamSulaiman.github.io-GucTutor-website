11.1 Introduction to Boolean Logic

Boolean logic, named after George Boole, is a branch of algebra that deals with values of true and false (often represented as 1 and 0). It is the mathematical foundation of digital circuits and computer programming logic. In Boolean logic, statements are either true or false, and operations are performed on these values to produce a true or false result.

Key Concepts:

  • Binary Values: True (1) and False (0).
  • Logical Operators: Operations that combine or modify Boolean values.
  • Truth Tables: Tables that show all possible input combinations for a logical operation and their corresponding outputs.

11.2 Basic Boolean Operators

There are three fundamental Boolean operators:

1. AND Operator:

  • Symbol: `AND`, `&`, `&&` (in programming)
  • Function: The result is True (1) only if ALL of its inputs are True (1). If any input is False (0), the result is False (0).
  • Truth Table:
    Input A Input B Output (A AND B)
    000
    010
    100
    111

2. OR Operator:

  • Symbol: `OR`, `|`, `||` (in programming)
  • Function: The result is True (1) if AT LEAST ONE of its inputs is True (1). The result is False (0) only if ALL inputs are False (0).
  • Truth Table:
    Input A Input B Output (A OR B)
    000
    011
    101
    111

3. NOT Operator:

  • Symbol: `NOT`, `!`, `~` (in programming)
  • Function: The result is the inverse of the input. If the input is True (1), the output is False (0), and vice versa. It is a unary operator (takes only one input).
  • Truth Table:
    Input A Output (NOT A)
    01
    10

11.3 Boolean Expressions

Boolean expressions are combinations of Boolean variables and operators that evaluate to either true or false. These expressions are used extensively in conditional statements (e.g., `if` statements) and loops in programming.

Examples:

  • `A AND B`
  • `NOT (A OR B)`
  • `(X AND Y) OR Z`

Boolean logic is the bedrock of digital computing, enabling computers to make decisions and perform complex operations by manipulating simple true/false values.
