Techniques of Limits (Continued)

  1. Direct Substitution
  2. If the limit results in an indeterminate form ($\frac{0}{0}$), "Undetermined":
    1. We must cancel the zero factor (i.e., identify common factors) by:
      1. Factorizing both numerator and denominator.
      2. Multiplying by the conjugate.
    2. Important rule: $\lim_{\theta\rightarrow0}\frac{sin~\theta}{\theta}=1$. This rule is also valid with "tan".
      • This is an important result.
      • Proof Example ($\lim_{x\rightarrow0}\frac{sin~2x}{sin~3x}$): $\lim_{x\rightarrow0}\frac{sin~2x}{sin~3x} = \lim_{x\rightarrow0}(\frac{sin~2x}{sin~3x}.\frac{x}{x}) = \lim_{x\rightarrow0}(\frac{sin~2x}{x}.\frac{x}{sin~3x})$ $= \lim_{x\rightarrow0}\frac{sin~2x}{x}\cdot \lim_{x\rightarrow0}\frac{x}{sin~3x}$ $= \frac{2}{3}\lim_{2x\rightarrow0}\frac{sin~2x}{2x}\cdot \lim_{3x\rightarrow0}\frac{3x}{sin~3x} = \frac{2}{3}$
      • Proof Example ($\lim_{x\rightarrow0}\frac{tan~2x}{sin~3x}$): $\lim_{x\rightarrow0}\frac{tan~2x}{sin~3x} = \frac{2}{3}\lim_{x\rightarrow0}(\frac{tan~2x}{2x}\cdot\frac{3x}{sin~3x})$ $= \frac{2}{3}\lim_{2x\rightarrow0}\frac{tan~2x}{2x}\cdot \lim_{3x\rightarrow0}\frac{3x}{sin~3x} = \frac{2}{3}\lim_{2x\rightarrow0}\frac{tan~2x}{2x}$ $= \frac{2}{3}\lim_{2x\rightarrow0}\frac{sin~2x}{2x}\cdot \lim_{x\rightarrow0}\frac{1}{cos~2x} = \frac{2}{3}$
  3. Squeezing "Sandwich" Theorem
    • We use it with Bounded functions ONLY.
    • Theorem 4: Suppose that $g(x)\le f(x)\le h(x)$ for all $x$ in some open interval containing $c$, except possibly at $x=c$ itself.
    • Suppose also that $\lim_{x\rightarrow c}g(x)=\lim_{x\rightarrow c}h(x)=L$.
    • Then $\lim_{x\rightarrow c}f(x)=L$.
    • Example 1: If $1-\frac{x^{2}}{2}\le u(x)\le1+\frac{x^{2}}{2}$, find $\lim_{x\rightarrow0}u(x)$.
      • Since, $\lim_{x\rightarrow0}(1-\frac{x^{2}}{2})=1=\lim_{x\rightarrow0}(1+\frac{x^{2}}{2})$, then (by the sandwich theorem) $\lim_{x\rightarrow0}u(x)=1$.
    • Example 2: Prove that: $\lim_{x\rightarrow0}x^{2}sin(\frac{1}{x})=0$.
      • Notice that, $-1\le sin(1/x)\le1$, so $-x^{2}\le x^{2}sin(1/x)\le x^{2}$.
      • As $\lim_{x\rightarrow0}x^{2}=0$ and $\lim_{x\rightarrow0}(-x^{2})=0$, therefore, by the sandwich theorem: $\lim_{x\rightarrow0}x^{2}sin(\frac{1}{x})=0$.
  4. Limits at infinity
    • When $x$ increases without bound ($x \rightarrow \infty$), the values of the function are very close to zero.
    • Similarly, when $x$ decreases without bound ($x \rightarrow -\infty$), the values of the function are very close to zero.
    • Technique for rational functions: To find the limit of a rational function as $x \rightarrow \pm\infty$, divide both the numerator and denominator by the highest order term in the denominator, then evaluate the limit.
    • Example: For $\lim_{x\rightarrow\infty}(2+\frac{sin~x}{x})=2$.
      • Since $-1 \le \sin x \le 1$, we have $-\frac{1}{x} \le \frac{\sin x}{x} \le \frac{1}{x}$ for $x > 0$.
      • As $\lim_{x\rightarrow\infty} -\frac{1}{x} = 0$ and $\lim_{x\rightarrow\infty} \frac{1}{x} = 0$, by the Sandwich Theorem, $\lim_{x\rightarrow\infty} \frac{sin~x}{x} = 0$.
      • Therefore, $\lim_{x\rightarrow\infty}(2+\frac{sin~x}{x})=2+0=2$.

One-Sided Limit (Motivation)

Consider the Heaviside (unit step) function defined as: $y(x)=\begin{cases}0,&x<0\\ 1,&x\ge0\end{cases}$

  • As $x\rightarrow0$ from the left ($x<0$), we can see that $y(x)\rightarrow0$. This is written as: $\lim_{x\rightarrow0^{-}}y(x)=0$.
  • As $x\rightarrow0$ from the right ($x>0$), we can see that $y(x)\rightarrow1$. This is written as: $\lim_{x\rightarrow0^{+}}y(x)=1$.
  • Therefore, there is no unique number that $y(x)$ approaches as $x\rightarrow0$. Thus, $\lim_{x\rightarrow0}y(x)=$ Does Not Exist "DNE".

Formal Definition of One-Sided Limit

  • We say that the limit of $f(x)$ equals $L$ as "$x \rightarrow a^{-}$" ($x$ approaches $a$ from left), if the values of $f(x)$ are arbitrarily close to a unique number $L$, when the values of $x$ are sufficiently close to $a$ ($x < a$), but not necessarily equal to $a$.
  • We say that the limit of $f(x)$ equals $M$ as "$x \rightarrow a^{+}$" ($x$ approaches $a$ from right), if the values of $f(x)$ are arbitrarily close to a unique number $M$, when the values of $x$ are sufficiently close to $a$ ($x > a$), but not necessarily equal to $a$.

When is it necessary to study left-hand and right-sided limits?

  1. If $f(x)$ is defined differently "with different expressions" around the limit study point.
  2. Whenever we have a modulus, and its symmetric point is the same as the limit study point.

Conclusion: If the function is defined on a closed interval $[a, b]$, then at point "$a$" we can only calculate the right-sided limit, and at point "$b$" we can only calculate the left-sided limit.
