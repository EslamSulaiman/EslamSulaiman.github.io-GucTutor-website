1. The Concept of Antiderivatives

We know that if $F(x)=x^2+c$, then $F^{\prime}(x)=2x$. Now, what if we have a function $f(x)=2x$? What function $F(x)$ could we differentiate to get $2x$? This reverse process is called finding the Antiderivative.

Definition: A function $F$ is an antiderivative of $f$ on an interval $I$ if $F^{\prime}(x)=f(x)$ for all $x$ in $I$.

Examples:

  • If $F(x)=x^{3}$, then $F^{\prime}(x)=3x^{2}$. So, $x^{3}$ is an antiderivative of $3x^{2}$.
  • Also $x^{3}+2$, $x^{3}-5$, $\sqrt{2}+x^{3}$ are antiderivatives of $3x^{2}$. This suggests that if $F(x)$ is an antiderivative of $f(x)$, then $F(x)+C$ (where $C$ is any constant) is also an antiderivative of $f(x)$.

Theorem 1: Functions with the same derivative differ by a constant.

If $F^{\prime}(x)=G^{\prime}(x)$ on an interval $I$, then $F(x)-G(x)=C$ for some constant $C$. So, $F(x)=G(x)+C$.

Important Result: If $F(x)$ is an antiderivative of $f(x)$, then the most general antiderivative of $f(x)$ is $F(x)+C$.

Examples: Find the general antiderivative for each function.

1. $f(x)=x^n$ (Power Rule for Antiderivatives)
Solution: $F(x)=\frac{x^{n+1}}{n+1}+C$, for $n\ne-1$. (If $n=-1$, $f(x)=x^{-1}=\frac{1}{x}$, then $F(x)=\ln|x|+C$).

2. $f(x)=\cos x$
Solution: $F(x)=\sin x+C$.

3. $f(x)=\sec^{2}x$
Solution: $F(x)=\tan x+C$.

4. $f(x)=e^{x}$
Solution: $F(x)=e^{x}+C$.

5. $f(x)=a^{x}$
Solution: $F(x)=\frac{a^{x}}{\ln a}+C$.

6. $f(x)=\frac{1}{\sqrt{1-x^{2}}}$
Solution: $F(x)=\sin^{-1}x+C$ or $-\cos^{-1}x+C$.

2. Indefinite Integral

The collection of all antiderivatives of a function $f$ is called the indefinite integral of $f$ with respect to $x$, and is denoted by $\int f(x) dx$. Thus, $\int f(x) dx = F(x)+C$, where $F^{\prime}(x)=f(x)$. The symbol $\int$ is an integral sign, $f(x)$ is the integrand, $dx$ identifies $x$ as the variable of integration, and $C$ is the constant of integration.

Properties of Indefinite Integrals:

  • Constant Multiple Rule: $\int k f(x) dx = k \int f(x) dx$ for any constant $k$.
  • Sum and Difference Rule: $\int [f(x)\pm g(x)] dx = \int f(x) dx \pm \int g(x) dx$.

Basic Integration Formulas

These are the reverse of the differentiation rules:

  • $\int x^{n} dx=\frac{x^{n+1}}{n+1}+C$, for $n\ne-1$.
  • $\int \frac{1}{x} dx=\ln|x|+C$.
  • $\int \sin x dx=-\cos x+C$.
  • $\int \cos x dx=\sin x+C$.
  • $\int \sec^{2}x dx=\tan x+C$.
  • $\int \csc^{2}x dx=-\cot x+C$.
  • $\int \sec x \tan x dx=\sec x+C$.
  • $\int \csc x \cot x dx=-\csc x+C$.
  • $\int e^{x} dx=e^{x}+C$.
  • $\int a^{x} dx=\frac{a^{x}}{\ln a}+C$.
  • $\int \frac{1}{\sqrt{1-x^{2}}} dx=\sin^{-1}x+C$.
  • $\int \frac{1}{1+x^{2}} dx=\tan^{-1}x+C$.
  • $\int \frac{1}{|x|\sqrt{x^{2}-1}} dx=\sec^{-1}x+C$.

Examples

Evaluate the following integrals:

1. $\int (x^{2}-2x+5) dx$
Solution: $\frac{x^{3}}{3}-x^{2}+5x+C$.

2. $\int (\sqrt{x}+\frac{1}{x^{2}}) dx$
Solution: $\int (x^{1/2}+x^{-2}) dx=\frac{x^{3/2}}{3/2}+\frac{x^{-1}}{-1}+C=\frac{2}{3}x^{3/2}-\frac{1}{x}+C$.

3. $\int \frac{x^{2}+x+1}{x} dx$
Solution: $\int (x+1+\frac{1}{x}) dx=\frac{x^{2}}{2}+x+\ln|x|+C$.

4. $\int (\cos x-2\sin x) dx$
Solution: $\sin x+2\cos x+C$.

5. $\int (5e^{x}+2^{x}) dx$
Solution: $5e^{x}+\frac{2^{x}}{\ln 2}+C$.

6. $\int \frac{x^{2}+1}{x^{2}} dx$
Solution: $\int (1+\frac{1}{x^{2}}) dx=\int (1+x^{-2}) dx=x-\frac{1}{x}+C$.

7. $\int \frac{3}{\sqrt{1-x^{2}}} dx$
Solution: $3\sin^{-1}x+C$.

Initial Value Problems

In many applications, we are given a function's derivative and asked to find the function. We will need to know a point on the curve of the function to be able to find the value of the constant of integration $C$. This kind of problem is called an Initial Value Problem.

Example: Find $f(x)$ if $f^{\prime}(x)=2x+1$ and $f(1)=3$.

Solution: $f(x)=\int (2x+1) dx = x^{2}+x+C$. Use the condition $f(1)=3$: $f(1)=1^{2}+1+C=3$ $2+C=3 \Rightarrow C=1$. Therefore, $f(x)=x^{2}+x+1$.
