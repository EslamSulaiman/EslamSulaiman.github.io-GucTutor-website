This lecture is a continuation of Chapter 1, "Basics: Types of Functions," from Lectures #2 and #3, and covers logarithmic functions, trigonometric functions, and a summary of domain rules.

6. Logarithmic Function

  • Definition: For any positive number $a \ne 1$, the function $f(x) = \log_a(x)$ is defined as the inverse function of $a^x$. This means that $y = \log_a(x)$ if and only if $x = a^y$.
  • Interpretation: $\log_a(x)$ can be thought of as "the power of '$a$' that gives $x$".
    • Example: $\log_2(32) = 5$, because $2^5 = 32$.
    • Examples: $\log_2(2) = 1$ and $\log_2(1) = 0$.
  • Graphs: The graph of $y = \log_a(x)$ is the reflection of $y=a^x$ about the line $y=x$.
  • Main Conclusions:
    • The domain of all logarithmic functions $\log_a(u)$ is $(u > 0)$.
    • $\log_a(1) = 0$ for any positive base 'a'.
  • Sketching Transformed Logarithmic Functions: To sketch a function like $f(x) = 1 - \log_2(x+3)$:
    1. Start with the basic graph of $\log_2(x)$.
    2. Reflect it across the x-axis (due to the negative sign before $\log_2$).
    3. Shift it left by 3 units (due to $x+3$).
    4. Shift it up by 1 unit (due to $+1$).
  • Special Cases:
    • Natural Logarithmic Function: Base is the number $e \approx 2.71828...$, denoted as $f(x) = \ln x$. Its graph is similar to other logarithmic functions with base > 1.
    • Common Logarithmic Function: Base is 10, denoted as $f(x) = \log x$.
  • Inverse Equations for $a^x$ and $\log_a x$:
    • $a^{\log_a x} = x$ (for $x > 0$).
    • $\log_a(a^x) = x$ (for all x).
    • Special cases for natural logarithm: $e^{\ln x} = x$ (for $x > 0$) and $\ln(e^x) = x$ (for all x).
  • Rules of Logarithms (for $x > 0$ and $y > 0$):
    1. **Product Rule**: $\log_a(xy) = \log_a x + \log_a y$ and $\ln(xy) = \ln x + \ln y$.
    2. **Quotient Rule**: $\log_a \frac{x}{y} = \log_a x - \log_a y$ and $\ln \frac{x}{y} = \ln x - \ln y$.
    3. **Reciprocal Rule**: $\log_a \frac{1}{y} = -\log_a y$ and $\ln \frac{1}{y} = -\ln y$.
    4. **Power Rule**: $\log_a x^y = y \log_a x$ and $\ln x^y = y \ln x$.
  • Change of Base Formula: A useful relation is available for changing the base of logarithms.

Summary for Domains

The domain rules for various function types were summarized:

  1. For sums, differences, or products of functions ($f \pm g$, $f \cdot g$), the domain is the intersection of their individual domains ($D_f \cap D_g$).
  2. For a polynomial function ($f$ is a polynomial), the domain is all real numbers ($\mathbb{R}$).
  3. For a rational function $f(x) = \frac{p(x)}{q(x)}$, the domain is all real numbers except where the denominator $q(x) = 0$.
  4. For $f(x) = \frac{1}{q^n(x)}$, the domain excludes values where $q(x) = 0$.
  5. For $f(x) = \sqrt{\Box}$, the domain requires the expression inside the square root to be greater than or equal to zero ($\Box \ge 0$).
  6. For $f(x) = \frac{1}{\sqrt{\Box}}$, the domain requires the expression inside the square root to be strictly greater than zero ($\Box > 0$).
  7. For an exponential function $f(x) = e^{\Box}$, the domain is the domain of $\Box$.
  8. For logarithmic functions $f(x) = \log_a(\Box)$ or $f(x) = \ln(\Box)$, the domain requires the argument to be strictly greater than zero ($\Box > 0$).

7. Trigonometric Functions

  • Definition of an Angle: An angle is formed by the rotation of a ray from its initial position (OA) to its terminal position (OB) about its initial point (O). Angles in standard position have the initial ray along the positive x-axis.
  • Measures of Angles: Angles can be measured in degrees (deg) or radians (rad).
    • A complete revolution is $360^\circ$ or $2\pi$ radians.
  • Conversion Formulas:
    • 1 degree = $\frac{\pi}{180}$ radians ($\approx 0.02$ radians). To convert degrees to radians, multiply by $\frac{\pi}{180}$.
    • 1 radian = $\frac{180}{\pi}$ degrees ($\approx 57$ degrees). To convert radians to degrees, multiply by $\frac{180}{\pi}$.
  • Definition for Acute Angle $\theta$: The six trigonometric functions (sine, cosine, tangent, cosecant, secant, cotangent) are defined as ratios of the lengths of sides of a right-angled triangle.
  • Periodic Functions: Trigonometric functions are periodic, meaning they repeat their values over a regular interval. The period is the smallest value 'p' such that $f(x+p) = f(x)$ for all x in the domain. The basic trigonometric functions like sine and cosine have a period of $2\pi$.
    • For functions like $\sin(ax+b)$ and $\cos(ax+b)$, the period is $\frac{2\pi}{|a|}$ and their graphs are shifted by $\frac{b}{a}$.
  • Basic Trigonometric Identities: For any angle $\theta$, $\sin^2\theta + \cos^2\theta = 1$.
  • ASTC-Rule (Quadrant Rule): This rule helps determine the sign (positive or negative) of trigonometric functions in each of the four quadrants.
    • All functions are positive in Quadrant I.
    • Sine is positive in Quadrant II.
    • Tangent is positive in Quadrant III.
    • Cosine is positive in Quadrant IV.
  • Finding Values of Non-Acute Angles: To find the value of a trigonometric function for a non-acute angle:
    1. Determine the sign based on the ASTC rule.
    2. Find the reference angle (the acute angle between the x-axis and the terminal side of the given angle).
    3. Use the sign and the trigonometric value of the reference angle to find the required value.

Inverse Trigonometric Functions

  • The lecture introduces inverse trigonometric functions, specifically the inverse sine function, $\sin^{-1}(x)$ or $\arcsin x$, which represents the angle whose sine is x.
  • $\sin x$ becomes 1-1 on a restricted domain, allowing for the definition of its inverse.
